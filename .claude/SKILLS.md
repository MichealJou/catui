# 项目推荐的 Skills

本项目推荐使用以下 Claude Skills 来提升开发效率：

## ✅ 已全局安装的 Skills

### Vue 生态
- **vue** - Vue 开发最佳实践
- **ant-design-vue** - Ant Design Vue 组件库
- **element-plus-vue3** - Element Plus Vue 3 组件库

### UI 设计
- **ui-design-system** - UI 设计系统模板
- **frontend-design** - 前端设计助手
- **interaction-design** - 交互设计指南

### 内置
- **ui-ux-pro-max** - UI/UX 设计智能助手（支持 50+ 风格，21 调色板）

---

## 安装命令

如需重新安装或更新这些 skills：

```bash
# Vue 生态
npx skills add bobmatnyc/claude-mpm-skills@vue -g
npx skills add teachingai/full-stack-skills@ant-design-vue -g
npx skills add teachingai/full-stack-skills@element-plus-vue3 -g

# UI 设计
npx skills add davila7/claude-code-templates@ui-design-system -g
npx skills add jwynia/agent-skills@frontend-design -g
npx skills add aj-geddes/useful-ai-prompts@interaction-design -g
```

---

## 使用说明

这些 skills 已全局安装，可在任何项目中使用。使用时：

1. 直接向 Claude 提问相关主题，skills 会自动触发
2. 或明确指定："使用 vue skill 帮我..."
3. UI 设计相关可直接使用 ui-ux-pro-max
